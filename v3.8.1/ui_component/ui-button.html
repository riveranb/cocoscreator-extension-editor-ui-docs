Loading State

<ui-button loading>loading...</ui-button>

Parameters

Attributes

// Button type
type: 'danger' | 'success' | 'primary' | 'warning' | 'default' | 'icon'

// Make background transparent and invert text and border colors
outline: boolean

// Disable the button
disabled: boolean

// Set the button to loading status
loading: boolean

Events

Available Events

'change'
'confirm' // general case use this event

Usage Examples

Vue.js

<ui-button @confirm="onConfirm">click me</ui-button>

// vue object
methods = {
    onConfirm() {
        this.clickedNumber++;
    }
};

Vanilla JavaScript

<ui-button id="button">click me</ui-button>

const element = document.getElementById('#button');
element.addEventListener('confirm', () => {
    clickedNumber++;
});
Note: This is a built-in UI component for Cocos Creator Editor. It can only be used within editor extensions and uses the editor's custom event system (e.g., `confirm` event instead of standard `click` event).
