<ui-drag-item hidden></ui-drag-item>

Attributes

value // drag item value type // drag item type, like: 'cc.Node' disabled readonly invalid hidden

Try It

Drag the item below to the drag area

Events

dragstart // when drag starts dragend // when drag ends

VUE JS

<ui-drag-item ref="dragItem" value="item1" type="cc.Node" @dragstart="onDragStart" @dragend="onDragEnd" ></ui-drag-item>

Vanilla JS

<ui-drag-item id="dragItem" value="item1" type="cc.Node"></ui-drag-item> const element = document.getElementById('#dragItem'); // addEventListener element.addEventListener('dragstart', (event) => { console.log('onDragStart'); }); element.addEventListener('dragend', (event) => { console.log('onDragEnd'); });
